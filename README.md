# ArkIDE tools
## Tools for people who want to work on or fork ArkIDE.

### What this includes
- Workspace setup script: A script that clones all of the repos needed to work on ArkIDE and removes the .git folders to refrain from having 20 initalised repositories in 1 folder.
- Branding Folder: A folder with all of the ArkIDE Branding.
